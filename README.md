# Hemsz Studio Website

A responsive, GitHub Pages-ready portfolio website for Hemsz Studio.

## Files

- `index.html` — page structure and content
- `style.css` — full responsive design and animations
- `script.js` — menu, reveal animations, filters and cursor effects
- `assets/` — place your logo and portfolio images here

## Preview in VS Code

1. Open this folder in VS Code.
2. Install the **Live Server** extension.
3. Right-click `index.html`.
4. Choose **Open with Live Server**.

## Publish on GitHub Pages

1. Create a GitHub repository.
2. Upload all files from this folder.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select `main` and `/ (root)`.
6. Save.

## Customise before publishing

- Replace the `H` placeholder mark with your real logo.
- Replace portfolio sample cards with your actual work.
- Add your WhatsApp number to the `https://wa.me/` link.
- Confirm your Instagram and email.
- Edit service descriptions and About text as needed.

The visual direction is original Afrofuturist royal-tech styling. It does not use Marvel or Black Panther copyrighted artwork, logos or characters.
